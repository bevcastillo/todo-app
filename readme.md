# TODO APP

# Cloning:
---
```sh
$ git clone https://github.com/rbo13/todo-app.git
```


# Installation:
---
```sh
$ npm install
```


# Running:
---
```sh
$ npm run server
```


# Creating database migration:
---
```sh
$ npm run migration:make
```


# Running database migration:
---
```sh
$ npm run migrate
```


# Creating seed file:
---
```sh
$ npm run seed:make
```


# Running seed data:
---
```sh
$ npm run seed:run
```
